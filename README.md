# Astro Commanders

Photon Fusion 기반 멀티플레이 PVP 전략 게임으로,
배치 시스템과 인게임 스탯 반영 구조를 구현했습니다.

---

## 개요

- 기간: 2026.02.09 - 2026.04.10
- 팀 구성: 기획 7명 / 개발 5명
- 플랫폼: Android (AOS)

---

(여기에 GIF 또는 이미지 추가)

- 배치 가능 영역 표시
- 카드 드래그를 통한 유닛 소환

## 핵심 기능

- 배치 가능 영역을 여러 구간 데이터로 분리하여 관리
- 전투 진행에 따라 배치 영역이 확장되는 구조 구현
- 유닛 소환 시 유저 스탯을 반영하는 시스템 구현
- 재소환 시간 감소 효과를 UI와 연동

---

## 기술적 특징

### 배치 시스템

배치 영역이 전투 진행에 따라 확장되어야 하는 요구사항이 있었고,
기존의 단일 영역 구조로는 이를 유연하게 처리하기 어려웠습니다.

이를 해결하기 위해 배치 영역을 여러 사각형 데이터로 분리하고,
상태에 따라 활성 영역을 동적으로 관리하는 구조로 개선했습니다.

그 결과 확장에 유연한 구조를 확보할 수 있었고,
플레이어에게 현재 배치 가능 영역을 시각적으로 명확하게 전달할 수 있었습니다.

---

### 스탯 반영 구조

아웃게임 성장 스탯을 인게임에 반영해야 했으나,
실시간으로 적용할 경우 동기화 및 일관성 문제가 발생할 수 있었습니다.

이를 해결하기 위해 유저의 최종 스탯을 별도로 관리하고,
유닛 생성 시점에 해당 스탯을 적용하도록 설계했습니다.

이를 통해 네트워크 환경에서도 안정적인 수치 적용이 가능해졌으며,
UI와 실제 게임 데이터 간의 일관성을 유지할 수 있었습니다.

## 프로젝트 설명

실시간 전략, 타워 디펜스, 로그라이트 요소를 결합한 PVP 게임으로,
증강 시스템을 통해 매 판 새로운 전략을 경험할 수 있도록 설계되었습니다.
