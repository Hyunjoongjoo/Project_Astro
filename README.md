## 개요

개발 기간: 2026.02.09 - 2026.04.10

팀 구성 : 기획 7명, 개발 5명

플랫폼 : 안드로이드 (AOS)

사용 기술 스택:

- C#
- Photon Fusion
- Firebase
- Unity Cloud Remote Config

## 게임 설명

- 매 판 새로운 전략, **실시간 증강 시스템**<br>
경기 중 무작위로 제시되는 3종의 증강 카드 중 하나를 선택해 영웅을 즉시 소환하거나 성능을 극한까지 끌어올릴 수 있습니다.
증강에 따라 미사일 개수가 늘어나는 등 파괴적인 위력을 발휘하며, 당신의 선택이 곧 승리로 가는 유일한 길이 됩니다.

- 협동과 경쟁을 동시에! **2 vs 2 대전모드**<br>
아군이 소환한 영웅에 내가 스킬 증강을 더해주거나, 아군에게 필요한 아이템을 전달할 수 있습니다.
팀원 간의 증강 공유와 연계를 통해 압도적인 화력을 경험할 수 있습니다.

- 플레이를 거듭할수록 강해지는 내 영웅들!<br>
전투를 통해 얻은 경험치와 골드로 영웅을 성장시킬 수 있습니다.
레벨이 증가할 수록 인게임에서 사용하는 영웅들의 능력치가 영구적으로 증가합니다.

## 게임 방법

- 처음 접속한다면 계정을 만드세요. 게스트 계정을 만든 후 설정에서 구글 연동을 할 수 있습니다.
- 로비에서 채팅 매크로와 영웅들의 정보를 확인해 보세요.
- 1vs1 혹은 2vs2 대전 중 원하는 모드를 선택하면 상대방을 찾습니다.
- 매칭에 성공하면 첫번째, 두번째 증강을 선택하세요. 영웅 추가, 스킬 강화, 아이템 추가를 할 수 있습니다.
- 전투에 돌입하면 하단의 패널에서 영웅 카드를 드래그하여 전장에 놓아 소환해 보세요.
- 적을 처치하면 증강 게이지를 획득하며, 이를 통해 전투력을 더욱 끌어 올리세요.
- 적의 함교를 파괴하면 승리합니다!
